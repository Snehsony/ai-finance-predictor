# AI Finance Predictor 🚀

This project is a real-world AI SaaS product that provides:

- Loan Prediction API
- Fraud Detection API
- Customer Churn Prediction

## Tech Stack
- FastAPI (Backend)
- Machine Learning (Scikit-learn)
- React (Frontend)
- Deployment (Render / Vercel)

## API Endpoints
- /predict/loan
- /predict/fraud
- /predict/churn

## Use Case
This system can be used by fintech companies to automate risk analysis and decision making.

## Future Scope
- Dashboard with analytics
- Real-time predictions
- Payment integration
